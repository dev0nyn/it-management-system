# IT Management System

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (Frontend & API routes)
- **Runtime**: [Node.js](https://nodejs.org/)
- **Database**: Interaction for all modules (Users, Assets, Tickets)
- **Styling**: Modern, responsive UI

---

## 🛠 Main Modules

### 👤 User Management
Manage organizational accounts, roles, and permissions securely. Ensure the right people have access to the right resources.

### 💻 Asset Management
Track hardware and software devices across the organization. Monitor lifecycle, assignments, and maintenance schedules.

### 🗳 Ticketing System
Efficiently handle IT issues with a structured support desk. Prioritize, assign, and resolve tickets to minimize downtime.

### 🌐 Network Monitoring
Proactive detection of network problems. Monitor health and performance to ensure 24/7 connectivity.

### 📊 Reports
Generate deep insights from system data. Visualize trends in asset utilization, ticket resolution times, and network stability.

---

## 🔔 Key Features

- **Centralized Dashboard**: A central hub where users select their work focus.
- **Secure Access**: Robust authentication and session management.
- **Automated Notifications**: Real-time alerts for IT staff regarding critical issues or new tickets.
- **Persistent Storage**: All actions and data are seamlessly synced with the database.

## ⌨️ Getting Started

### Prerequisites
- Node.js (latest LTS)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
